# Contributing

Thank you for considering contributing to RubikaBotPHP.

We welcome all contributions including bug fixes, documentation improvements and new features.

---

## Requirements

* PHP 8.1+
* Composer
* Git

---

## Installation

```bash
git clone https://github.com/Sobhansgh/Rubikabotphp.git

cd Rubikabotphp

composer install
```

---

## Create a Branch

```bash
git checkout -b feature/my-feature
```

---

## Coding Style

Please follow:

* PSR-1
* PSR-4
* PSR-12

---

## Running Tests

```bash
composer test
```

---

## Commit Messages

Examples

```text
Add webhook router

Fix file upload

Improve documentation
```

---

## Pull Requests

Please ensure:

* Tests pass
* Code follows PSR standards
* Documentation is updated
* No unnecessary files are included

---

Thank you ❤️
